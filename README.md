# Wallpaperplay
A small script written in shell scripting that automates the download process of wallpapers from https://www.wallpaperplay.com

#### How to run?
- ./wallpaper.sh [url]

#### Example:-
- ./wallpaper.sh https://wallpaperplay.com/board/4k-minimalist-wallpapers
